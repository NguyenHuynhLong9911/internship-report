### Week 7 Objectives

* Define the project idea for **SmartStudy AI - an intelligent learning assistant on AWS**.
* Identify the main users, core problems, and expected learning features.
* Draft an initial AWS architecture for a Generative AI and RAG-based learning platform.
* Prepare the initial project scope before implementation.

### Project Description

SmartStudy AI is an intelligent learning platform that helps students study more effectively through Generative AI and Retrieval-Augmented Generation (RAG). Users can upload learning materials, interact with an AI chatbot, generate quizzes automatically, and receive learning result evaluations directly on the web platform.
